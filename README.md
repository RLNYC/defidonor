# Defi Donors
The Dapp allows donors transferring crypto assets into their donation fund account and receiving tax benefits.

- Live Site: https://deifdonors.netlify.app/  (Works in Rinkeby Test Network)

## Techologies
- React
- Bootstrap 4
- Solidity
- Openzeppelin
- nft.storage
- BitGo SDK
- Chainlink
- Covalent API
- Gnosis
